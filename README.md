# amoCRM Promotions Widget

## Описание
Виджет для amoCRM для применения акций и скидок в карточке сделки.

## Структура
- browser — версия для локальной разработки
- amocrm — версия для загрузки в amoCRM
- shared — общая бизнес-логика

## Демо
Browser-версия запускается открытием `browser/index.html`

## amoCRM
Для установки загрузите содержимое папки `amocrm` как виджет.
